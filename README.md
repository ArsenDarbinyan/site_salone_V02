# Նազել Սալոն - Պրեմիում Գեղեցկության Սալոն

## Описание
Премиальный одностраничный сайт для салона красоты в Армении. Оптимизирован для мобильных устройств.

## Цветовая схема
- Rose Gold: #B76E79
- Silver: #C0C0C0
- Black: #1a1a1a
- Фон: #fafafa

## Структура проекта

```
mysite/
├── index.html          # Основной файл сайта
├── logo.png            # Логотип (замените на свой)
├── salon.jpg           # Фото салона (замените на своё)
├── master1.jpg         # Фото мастера 1 (замените на своё)
├── master2.jpg         # Фото мастера 2 (замените на своё)
├── master3.jpg         # Фото мастера 3 (замените на своё)
├── master4.jpg         # Фото мастера 4 (замените на своё)
└── README.md           # Этот файл
```

## Как изменить данные

### 1. Логотип
Замените файл `logo.png` на ваш логотип. Рекомендуемый размер: 50x50px для навигации, 60x60px для футера.

### 2. Фото салона
Замените файл `salon.jpg` на фото вашего салона. Рекомендуемый размер: 280x350px.

### 3. Фото мастеров
Замените файлы `master1.jpg`, `master2.jpg`, `master3.jpg`, `master4.jpg` на фото ваших мастеров.
Рекомендуемый размер: 150x150px (круглые).

Или измените пути в массиве `masters` в JavaScript:

```javascript
const masters = [
    {
        id: 1,
        name: 'Աննա Պետրոսյան',
        photo: 'path/to/your/photo.jpg', // Измените путь
        specialization: 'Մատնահարդար և վարսահարդար',
        // ...
    }
];
```

### 4. Добавление нового мастера

В JavaScript найдите массив `masters` и добавьте новый объект:

```javascript
{
    id: 5, // Уникальный ID
    name: 'Имя Фамилия',
    photo: 'master5.jpg',
    specialization: 'Специализация',
    services: [1, 2], // ID услуг, которые выполняет мастер
    servicePrices: [
        { name: 'Услуга 1', price: '10000 - 15000 դրամ' },
        { name: 'Услуга 2', price: '15000 - 20000 դրամ' }
    ]
}
```

### 5. Изменение услуг

В JavaScript найдите массив `services`:

```javascript
const services = [
    {
        id: 1,
        name: 'Մատնահարդարում',
        icon: 'fa-hand',
        description: 'Описание услуги',
        items: [
            { name: 'Конкретная услуга', price: 'от X до Y դրամ' }
        ]
    }
];
```

### 6. Изменение отзывов

В JavaScript найдите массив `reviews`:

```javascript
const reviews = [
    {
        text: 'Текст отзыва',
        author: 'Имя Фамилия',
        initials: 'ИФ' // Первые буквы имени и фамилии
    }
];
```

### 7. Изменение контактов

В HTML найдите секцию `<footer>` и измените:

```html
<p><i class="fas fa-map-marker-alt"></i> Ваш адрес</p>
<p><i class="fas fa-phone"></i> Ваш телефон</p>
<p><i class="fas fa-envelope"></i> Ваш email</p>
```

### 8. Изменение социальных сетей

В футере измените ссылки:

```html
<a href="https://instagram.com/yourprofile" class="social-icon">
    <i class="fab fa-instagram"></i>
</a>
<a href="https://facebook.com/yourpage" class="social-icon">
    <i class="fab fa-facebook-f"></i>
</a>
```

## Иконки

Сайт использует Font Awesome 6.4.0. Доступные иконки:
- https://fontawesome.com/icons

Для изменения иконки услуги измените `icon` в массиве `services`:
- `fa-hand` - рука
- `fa-foot` - нога
- `fa-scissors` - ножницы
- `fa-bolt` - молния/лазер
- И другие из Font Awesome

## Запуск сайта

Просто откройте файл `index.html` в браузере.

Для локального сервера можно использовать:
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server
```

Затем откройте http://localhost:8000

## Особенности

- **Mobile-first**: Оптимизирован для мобильных устройств
- **Адаптивный**: Работает на всех размерах экранов
- **Модульный код**: Все данные в отдельных массивах
- **Анимации**: Плавные переходы и эффекты
- **Фильтрация**: Мастера фильтруются по услугам
- **Интерактивные карточки**: Раскрываются при клике
